# Product strategy

## Positioning
Do not market this as "another landing template".
Market it as:
- premium landing system
- launch infrastructure
- developer-first commercial web kit

## Core differentiators
1. restrained premium aesthetics
2. section-first architecture
3. documentation and tokens
4. multi-archetype demos
5. faster shipping and clearer sales positioning

## Pricing suggestion
- Starter: $49
- Pro: $99
- Lifetime: $149

## Target users
- web developers
- indie hackers
- founders
- agencies
- portfolio builders
